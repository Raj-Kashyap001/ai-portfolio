# **App Name**: Digital Oasis

## Core Features:

- Content Display: Display blog posts, projects, and other content from remyjouni.dev in a clear and engaging format.
- Project Portfolio: Showcase projects with descriptions, images, and links to live demos or code repositories.
- Blog Section: Display blog posts with categories, tags, and a search function.
- Contact Form: Allow visitors to send messages directly to the site owner.
- Dynamic Content Generation: Utilize generative AI tool to automatically generate summaries or highlights for each project or blog post to keep the homepage fresh and relevant.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey professionalism and creativity.
- Background color: Light Gray (#F0F0F0) for a clean, modern look.
- Accent color: Soft Lavender (#E6E6FA) to add a touch of elegance and highlight key elements.
- Body and headline font: 'Inter' for a clean and modern reading experience.
- Use minimalist icons to represent different sections and categories.
- Employ a grid-based layout for a structured and visually appealing presentation of content.
- Subtle animations, such as fade-ins or transitions, to enhance user experience.
